    Key Commands

    note for windows:
    cmnd = cntrl
    opt = alt

    Chrome:
    Cmnd+opt+i open dev tools cmnd+shift+r hard refresh

    Emmet:
    Command + option + ↓ = selects multiple lines of code for multi edit.
    Option + Shift + ↓ = duplicates the current line

    Finder:
    Command + Tab = tabs through open apps
    Command + Tab + Shift = tabs backwards through open apps
